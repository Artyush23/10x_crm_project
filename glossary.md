# Technical Glossary

## 1. Authentication

**English:** Authentication is the process of checking that a user is who they claim to be.

**ქართული:** ავთენტიკაცია ნიშნავს მომხმარებლის მონაცემების შემოწმებას, მაგალითად ელფოსტისა და პაროლის შედარებას.

## 2. Session

**English:** A session represents the currently signed-in user while they use an application.

**ქართული:** სესია ინახავს ინფორმაციას, თუ რომელი მომხმარებელია ახლა სისტემაში შესული.

## 3. Validation

**English:** Validation checks whether entered data follows the required rules before it is accepted.

**ქართული:** ვალიდაცია ამოწმებს, სწორად არის თუ არა შევსებული ველები მონაცემების შენახვამდე.

## 4. Fetch

**English:** Fetch is a browser API used to send network requests and receive responses.

**ქართული:** `fetch` ბრაუზერის ფუნქციაა, რომლითაც სერვერიდან მონაცემებს ვითხოვთ ან ვაგზავნით.

## 5. Endpoint

**English:** An endpoint is a specific URL where an API provides or accepts data.

**ქართული:** ენდფოინტი არის API-ის კონკრეტული მისამართი, საიდანაც მონაცემებს ვიღებთ ან სადაც ვაგზავნით.

## 6. Request Method

**English:** A request method describes the intended action of an HTTP request, such as GET, POST, or DELETE.

**ქართული:** მოთხოვნის მეთოდი აჩვენებს მოქმედებას: GET იღებს, POST ამატებს, ხოლო DELETE შლის მონაცემს.

## 7. JSON

**English:** JSON is a text format commonly used to exchange structured data between systems.

**ქართული:** JSON არის ტექსტური ფორმატი, რომელშიც ობიექტებისა და სიების სახით ვინახავთ და გადავცემთ მონაცემებს.

## 8. State

**English:** State is the current data that determines what an application displays and how it behaves.

**ქართული:** მდგომარეობა არის აპის მიმდინარე მონაცემები, მაგალითად კლიენტების სია ან არჩეული ფილტრი.

## 9. Event Listener

**English:** An event listener runs a function when an event such as a click, input, or form submission occurs.

**ქართული:** მოვლენის მსმენელი ელოდება მომხმარებლის მოქმედებას და შემდეგ შესაბამის ფუნქციას უშვებს.

## 10. Deployment

**English:** Deployment is the process of publishing an application so people can access it online.

**ქართული:** დეპლოიმენტი ნიშნავს დასრულებული აპის ინტერნეტში ატვირთვას და საჯაროდ ხელმისაწვდომად გახდომას.
